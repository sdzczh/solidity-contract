## 4. LPDividendUsdt，加LP分红
加 LP 分红 USDT，推荐关系绑定，推荐分红本币，营销钱包，限购，自动杀区块机器人
买卖14%滑点，3%给加LP池子分红，4%分配10级推荐，1%营销钱包，1%备用拉盘，5%进入NFT盲盒
推荐分红，1级0.48%,2级0.44%,3级0.42%，4-10级各0.38%
限购总量 1%
